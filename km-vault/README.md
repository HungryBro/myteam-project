# KM Vault (Knowledge Management)

This is the central repository for all knowledge, research, and logs generated by the trading team. It is structured to be compatible with Obsidian.

## Structure

- **daily-notes/**: Daily logs and quick notes from agents.
- **research/**: In-depth market research.
    - **crypto/**: Crypto-specific research.
    - **forex/**: Forex-specific research.
    - **youtube-summaries/**: Summaries of videos from designated channels.
- **trade-logs/**: Detailed logs of all executed trades.
    - **crypto/**: Binance trade logs.
    - **forex/**: MT5 trade logs.
- **strategies/**: Documentation of trading strategies and backtest results.
- **weekly-reviews/**: Weekly performance reviews and strategic adjustments.

## Usage
All agents have read/write access to this vault. Agents should use the provided templates in their memory folders when creating new files here.
