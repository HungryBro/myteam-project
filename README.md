# myteam-project

**คำอธิบาย:** OpenClaw Multi-Agent Trading Bot (Forex + Crypto)

**3 Agents:** Chief, Analyst, Executor

**Tech stack:** OpenClaw, TradingView MCP, MT5, Discord, n8n, OpenRouter

## Quick Start

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/HungryBro/myteam-project.git
    cd myteam-project
    ```

2.  **Copy and configure environment variables:**
    ```bash
    cp .env.example .env
    # Fill in your API keys and configurations in the .env file
    ```

3.  **Start the services:**
    ```bash
    make up
    ```
